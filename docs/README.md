# ScratchScope Lottery Intelligence

ScratchScope is a local-first, mobile-first browser PWA for organizing cached North Carolina lottery public data, comparing scratch-offs, reviewing draw snapshots, saving random quick picks, and setting responsible-play budgets.

## Run Locally

```powershell
npm.cmd run dev
```

Open the local URL printed by the server, usually `http://127.0.0.1:5173/`.

## iPhone Home Screen

Open the app in Safari, tap Share, then choose **Add to Home Screen**. The PWA manifest uses the short name `ScratchScope`.

## Important

The included lottery data is a dated cache, not a live feed. Verify current games, prizes, jackpots, draw times, and results on the official NC Lottery site. ScratchScope never guarantees wins, predicts numbers, or improves odds.
