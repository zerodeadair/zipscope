# Data Sources

Current build:

- Scratch-off games: dated cache from official NC Lottery game and prizes-remaining pages.
- Draw games: dated cache; every draw value is labeled for official verification.
- Winner locations: sourced NC Lottery winner articles with approximate map coordinates.
- Cash Pop and Carolina Keno: hidden from active dashboards until a reliable connector is added.
- Saved picks and preferences: user-entered local browser data.

Automatic refresh still needs a same-origin backend connector or published official API. The UI shows unavailable states instead of substituting sample records.
