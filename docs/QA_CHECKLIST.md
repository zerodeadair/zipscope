# QA Checklist

- Mobile dashboard is glanceable.
- Bottom navigation works.
- Scratch-Off Kiosk filters by price and search.
- Ticket detail sheet opens and closes.
- Compare tray works.
- Ticket logging adds an unchecked ticket.
- Profile switcher separates Billy and Cousin tickets.
- Responsible-play toggles persist locally.
- Export backup downloads JSON.
- Daytime theme is the first-load default.
- Night theme persists only after explicit selection.
- Unavailable games show Coming Soon and do not appear in active tabs.
- Empty filters show a clear reset action.
- Copy, export, compare, modal close, and saved-number check actions provide feedback.
- Cached/live-data labels are visible and no demo retailer signals are rendered.
- Build completes with `npm.cmd run build`.
