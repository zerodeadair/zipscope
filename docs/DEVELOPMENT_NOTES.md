# Development Notes

The first version is a dependency-free browser PWA with no paid API requirements. It emphasizes a touch-first dashboard, scratch-off kiosk, ticket wallet, profile-separated history, retailer planning, and responsible-play controls.

Use `npm.cmd` on this Windows machine if PowerShell blocks `npm.ps1`. The package has no external dependencies, so `npm.cmd run dev` and `npm.cmd run build` work without an install step.
